### 布局规范

| 区域 | 尺寸 | 说明 |
|------|------|------|
| 侧边栏 | 240px | 固定宽度，响应式可折叠 |
| 顶栏 | 56px | 品牌 + 状态指示 + 设置入口 |
| 状态栏 | 40px | 底部，当前书籍统计信息 |
| 主内容 | 剩余 | 按页面类型自适应 |

| # | 页面 | 路由 | 布局 | 优先级 |
|---|------|------|------|--------|
| 1 | 仪表盘 | `/` | 单栏 + 卡片网格 | P0 |
| 2 | 书籍详情 | `/book/:id` | 双栏 | P0 |
| 3 | 章节写作 | `/book/:id/chapter/:num` | 三栏 | P0 |
| 4 | 故事设定 | `/book/:id/bible` | 双栏 | P0 |
| 5 | 新建书籍 | `/book/new` | 居中表单 | P1 |
| 6 | LLM 配置 | `/settings` | 居中卡片 | P1 |
| 7 | 导出 | `/export` | 居中卡片 | P2 |

## 五、交互设计要点

| 交互 | 行为 |
|------|------|
| 写下一章 | 点击 -> 按钮变为 loading -> 分步显示"规划中>撰写中>审计中" -> 完成后显示结果 |
| 保存草稿 | Ctrl+S / 点击保存 -> 自动保存至本地文件系统 |
| 审计触发 | 自动(写完后) + 手动(点击审计按钮) |
| 章节状态切换 | 审计通过 -> 待审阅 -> 人工点击"通过"后变为已通过 |
| 侧边栏折叠 | 小屏幕自动折叠，可通过汉堡菜单展开 |
| 主题 | 亮色/暗色模式 |
